# Sistema de Gestión de Citas Médicas - Clínica Serkeo

Proyecto final desarrollado en lenguaje **C** para la gestión integral de una clínica.

## Estructura del Proyecto
- **admin.c / .h**: Control del menú principal y gestión de usuarios.
- **citas.c / .h**: Lógica para agendar, verificar y cancelar citas.
- **medicos.c / .h**: Registro de doctores y diagnósticos.
- **pacientes1.c / .h**: Módulo para que los pacientes vean su historial.
- **validaciones.c / .h**: Funciones de seguridad y validación de cédulas.

## Requisitos
- Compilador GCC o entorno Zinjai.
- Librería estándar `stdlib.h` para funciones de sistema.
